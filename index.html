I'm a software engineer with multiple years of experience building full-stack applications using AWS, Node.js, Ruby, JavaScript/ES6+, PostgreSQL, Rails, and React. Most recently, I built Haven, a secrets manager.

Haven is an open-source developer tool for managing your application secrets. Built using Node.js and Amazon Web Services, it is easy to set up and integrate with your Node applications. It protects your secrets through encryption, access control, and injection at runtime. Haven also offers both a UI and a CLI to help make secrets management easy.
